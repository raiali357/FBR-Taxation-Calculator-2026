# Pakistan Withholding Tax Calculator TY 2026

Static, GitHub Pages ready withholding tax calculator based on the KPMG Taseer Hadi & Co. Withholding Tax Collection / Deduction Rate Card for Tax Year 2026.

## Included calculators

- Universal transaction withholding calculator
- Salary tax under section 149 with surcharge option
- Rent of immovable property under section 155
- Immovable property sale and purchase under sections 236C and 236K
- Motor vehicle registration and annual/lump-sum tax under sections 231B and 234
- Electricity, cash withdrawal, card remittance, bonus shares and common WHT categories
- Searchable reference table

## Publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files in this folder.
3. Go to `Settings > Pages`.
4. Select `Deploy from a branch`.
5. Choose the `main` branch and `/root` folder.
6. Open the Pages URL after GitHub finishes deployment.

## Source

The PDF rate card is included as `KPMG-Withholding-Tax-Rate-Card-TY-2026.pdf`.

This tool is an estimate for working papers and client discussion. Confirm special exemptions, reduced rates, final/minimum/adjustable treatment and legal updates before filing.
